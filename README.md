# Sipariş Takip Uygulaması

Bu uygulama, restoran/kafe gibi işletmelerde masa ve sipariş takibini kolaylaştırmak için geliştirilmiş bir web uygulamasıdır.

## Özellikler

- Masa durumu takibi (Aktif/Boş)
- Sipariş ekleme ve silme
- Masa bazlı toplam hesap görüntüleme
- Dinamik masa yönetimi

## Kurulum

1. Projeyi klonlayın:
```bash
git clone [REPO_URL]
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Uygulamayı başlatın:
```bash
npm start
```

## Teknolojiler

- React
- Material-UI
- Redux Toolkit
